Проект "Пивные Танки" - это консольная игра в жанре тактического симулятора танковых сражений времен Второй мировой войны.

## Описание

Игрок выбирает один из трех доступных танков и отправляется на поле боя в Берлине. Цель игры - успешно выполнить боевые задачи на выбранной линии фронта, уничтожая вражескую технику и пехоту противника.

## Особенности

- **Три уникальных танка** с различными характеристиками:
  - Panther D (высокий урон, среднее здоровье)
  - T34-85 (сбалансированные характеристики)
  - M4 Sherman (высокое здоровье, низкий урон)

- **Тактические сражения** с выбором целей и стратегий
- **Реалистичная механика повреждений** и состояния техники
- **Различные типы противников**:
  - Вражеские танки (Marder II, Pz.Kpfw. III)
  - Пехотные подразделения (штурмовики, гренадеры)
  - Артиллерийские орудия (гаубицы)

## Требования

- .NET Framework
- Консоль с поддержкой русского языка

## Управление

Игра полностью управляется через консольный ввод цифровых команд. В процессе игры игрок выбирает действия через нумерованные меню:

- Выбор танка и карты
- Управление двигателем танка
- Выбор целей в бою
- Тактические решения

## Структура проекта

Проект организован в несколько пространств имен:

- `Play.Tanks` - классы танков и их характеристики
- `Play.Maps` - игровые карты
- `Play.Enemys` - вражеские юниты

## Геймплей

1. Выбор танка с уникальными характеристиками
2. Выбор линии фронта на карте Берлина
3. Тактические сражения с различными типами противников
4. Управление состоянием танка (двигатель, повреждения)
5. Стратегический выбор целей в многопозиционных боях
